## 特性

- ⚡ 开箱即用, 完全实现中间件零依赖
   - 默认使用Sqlite, 也可自由切换PostgreSQL、MySQL
   - 默认使用文件缓存, 支持切换为Redis
- 🚢 支持裸机安装和云原生的方式安装常用软件
- 🚀 通过扩展ansible的自动化功能, 大大提高软件部署效率
